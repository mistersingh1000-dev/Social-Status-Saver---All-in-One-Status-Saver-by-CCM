# Social Status Saver - All in One Status Saver by CCM

  A complete mobile application built with Expo React Native that allows users to legally save and manage social media status videos and images.

  ## Features
  - **WhatsApp Status Viewer** - View and save visible WhatsApp statuses
  - **Link Downloader** - Download media from Instagram, Facebook, Snapchat via public URLs
  - **Saved Media Gallery** - Browse all saved media with platform and type filters
  - **Privacy Compliant** - Only accesses publicly available content

  ## Tech Stack
  - **Frontend**: Expo React Native with Expo Router (file-based routing)
  - **Backend**: Express.js (TypeScript)
  - **Storage**: AsyncStorage for local persistence
  - **UI**: Custom components with Ionicons, Inter font, dark/light theme

  ## Getting Started

  1. Install dependencies:
  ```bash
  npm install
  ```

  2. Start the backend:
  ```bash
  npm run server:dev
  ```

  3. Start the frontend:
  ```bash
  npx expo start
  ```
